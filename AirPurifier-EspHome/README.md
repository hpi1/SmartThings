SmartThings driver for Xiaomi Air Purifier 3C.

Device Wifi module needs to be flashed with EspHome.

Supported features:
 - switch
 - fan speed
 - PM2.5 sensor
 - Filter status
 - Device health

Add device from App ("Add Device" -> "Search Nearby Devices").

If device is not found, try power cycling air purifier. Sometimes
it can be discovered only within few minutes after it has been
powered on.
